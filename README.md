## 🚀 Features

- **Employee Activity Log**: Audit employee actions with flexible filtering.
- **Employee Summary**: Statistical overview of workforce status.
- **Activity Report**: Analytical report of system usage patterns.
- **Contact Search**: Efficient search with built-in pagination.
- **Workflow Participants**: Tracking participants in business processes.
- **Login History**: Monitoring user engagement and security.
- **Activity Dashboard**: 360-degree view of employee performance.

## 🛠️ Technology Stack

- **Database**: SQL Server
- **Language**: T-SQL (Stored Procedures)
